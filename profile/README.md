**类别：** 工具 / 教育

一款面向开发者的低门槛创意实现平台，让你用熟悉的 **JavaScript** 和 **React**，快速开发 MVP 原型。🚀

红糖云服提供从 0 到 1 的一站式路径：  
**学习 → 开发 → 发布 → 分享**，一气呵成。

只需一台 iOS 设备和一台电脑，即可开始使用 Web 技术构建项目。  
作为开发者工具，你可以使用 JavaScript 和 React 创建具有交互式手势与图形的体验。

> ✨ 注意：建议具备一定编程经验  
> ✨ 技术规格：此版本使用 React Native 0.83.4

* 🎓 多套开发模板，开箱即用、快速立项
* 📱 支持 ✨ [React Native](https://reactnative.dev/) 开发跨端应用（iOS / Android / H5 / 红糖云服小程序）
* 🌐 支持 Web 技术快速构建 MVP 原型并验证创意
* 🎮 支持 ✨ [Godot](https://godotengine.org/zh-cn/) 开发小游戏（iOS / Android / H5 / 微信 / 红糖云服小游戏）
