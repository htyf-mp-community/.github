# 红糖云服App 👋

![](./header_1.png)

一款可独立开发、独立部署、独立更新、无需审核的小程序开发测试容器. 🌈    

* 🎓   可使用多套开发模板
* 🌱   支持 ✨ [Taro](https://docs.taro.zone/docs/) 开发多端小程序
* 📚   支持 ✨ [React-native](https://reactnative.dev/) 开发更完善的应用
* 💻   支持 ✨ [cocos-creator](https://www.cocos.com/creator) 开发小游戏
